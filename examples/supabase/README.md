# Supabase 

## Getting Started

First, clone the repository and navigate to this example:

```bash
git clone https://github.com/langdb/langdb-samples.git
cd langdb-samples/examples/supabase
```

## Requirements

```bash
pip install supabase openai 
``` 

## Setup

### ENV Variables

Put it in a `.env` file

```
LANGDB_API_KEY="xxx"
LANGDB_PROJECT_ID="xxx"
SUPABASE_URL="xxx"
SUPABASE_API_KEY="xxx"
```
